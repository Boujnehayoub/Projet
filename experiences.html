Projet de Fin d'Études

SIFAST - Sfax, Tunisie
Février 2025 - Mai 2025
Système de Monitoring Intelligent pour la Sécurité et la Performance des Sites Web

Dans un contexte où les cybermenaces deviennent de plus en plus sophistiquées, ce projet a consisté à concevoir et développer une plateforme intelligente de surveillance des sites web, combinant détection en temps réel des menaces et optimisation des performances.

Fonctionnalités clés
  • Analyse des logs en temps réel
  • Détection des comportements anormaux
  • Génération d'alertes de sécurité
  • Surveillance des performances serveurs
  • Dashboard interactif
  • Recommandations personnalisées par IA
Technologies utilisées
Next.js Flask MongoDB Python Docker GitHub Postman VS Code PyCharm

SIFAST - Sfax, Tunisie
Juillet 2024 - Août 2024
Intégration de l'IA dans une Application Web

Développement d'un chatbot intelligent intégré à un plugin WordPress pour aider les utilisateurs à rechercher des résidences étudiantes à partir d'un dialogue interactif. Le système exploite une API REST pour récupérer dynamiquement les informations (ville, prix, services, image, etc.) et propose une expérience utilisateur fluide.

Réalisations
  • Développement plugin WordPress personnalisé
  • Création d'une API REST dynamique
  • Intégration chatbot avec logique NLP
  • Interface utilisateur responsive
  • Déploiement avec Docker
Technologies utilisées
WordPress PHP JavaScript MySQL HTML/CSS Docker Postman GitHub Slack
